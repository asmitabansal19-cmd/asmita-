The website is deployed on GitHub Pages and can be accessed at:
**https://asmitabansal19-cmd.github.io/asmita-/**

## 🚀 GitHub Pages Setup

To deploy this website on GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Set source to "Deploy from a branch"
4. Select branch: `main`
5. Select folder: `/ (root)`
6. Click Save

Your site will be live at: `https://asmitabansal19-cmd.github.io/asmita-/`

## 📸 Adding Images

Create an `assets/` folder and add the following images:

- `hero-candles.jpg` - Hero section candles
